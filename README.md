# Terminal JSON Editor (In-Progress)

A terminal-based JSON editor for Linux systems (including headless environments), designed to allow easy navigation, viewing, and modification of JSON data in a tree-like structure. This project is currently in active development.

# Features (Planned / In-Progress)

- Parse any valid JSON into a C-based tree structure.
  
- Navigate nested objects and arrays via terminal interface.
  
- Edit values, add or remove keys/nodes.
  
- Save modifications back to JSON.
  
- Lightweight and headless-friendly (no GUI required).
  

# Supported Platforms

- Linux (terminal)
  
- Linux headless environments
  
- Potential future support for macOS terminal (TBD)
